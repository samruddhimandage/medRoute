# MedRoute 

An emergency response web application that helps users quickly find the most suitable nearby hospital and the fastest route during a medical emergency.

## Problem Statement

During emergencies, people often waste valuable time searching for hospitals, ambulance contacts, and directions.

MedRoute simplifies this process by helping users reach the right hospital as quickly as possible.

## Features

* 📍 Automatic location detection
* 🎤 Voice-based symptom input
* 📝 Text-based emergency description
* 🏥 Smart hospital recommendations based on emergency type
* 🚑 Ambulance contact information
* 🗺️ Fastest route and navigation support
* 🌐 Multi-language support

  * English
  * Hindi
  * Marathi
  * Tamil
  * Telugu
  * Kannada
*  AI-assisted symptom categorization

## How It Works

1. Detects the user's current location.
2. Accepts symptoms through voice or text.
3. Identifies the type of emergency.
4. Recommends nearby hospitals.
5. Displays ambulance contact options.
6. Shows the fastest route to the selected hospital.
